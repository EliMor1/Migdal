

# A simple project purposed to display the governmental car garage listed, adding new garages to the data store and maintain existing car garages.

## Table of Contents

- [Installation](#installation)
- [Features](#features)
- [Usage](#usage)

## Installation

To install all dependencies, navigate to root directory and run npm install both in client and server directories
To run the project -
* run npm run start in the server side (NestJS) a framework for Node.js
* run ng serve in the client side (Angular), navigate in your browser to [localhost:4200](http://localhost:4200).

## Features

The project contains few main features:

* Update / Create a new Car Garage manually and add the garage to the list (displayed on a table in the main page) and insert the newly created garage in the data store.
* Create a new car garage or garages chosen by a select box and insert them into our data store.
* View details about a chosen car garage -> navigate to Update section or Head back to the table (main page).

## Usage

All the features may be available upon using the correct environment variables.
